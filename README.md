# Site Vitrine — Projet étudiant

## Structure du projet

```
projet-vitrine/
├── index.html          ← Page principale (à modifier)
├── css/
│   ├── style.css       ← Styles principaux (couleurs, composants, layout)
│   └── responsive.css  ← Adaptation mobile (media queries)
├── js/
│   └── main.js         ← Interactivité (formulaire, localStorage, navbar)
└── images/
    └── placeholder.txt ← Lis ce fichier pour savoir quelles images ajouter
```

## Par où commencer ?

1. Ouvre `index.html` et cherche tous les `[MODIFIE ICI]`
2. Remplace le texte de présentation par les vraies infos
3. Ajoute tes images dans le dossier `images/`
4. Change les couleurs dans `css/style.css` → section `:root`
5. Ouvre le site dans le navigateur et teste le formulaire

## Comment tester le formulaire ?

Le formulaire sauvegarde les commentaires dans le **localStorage** du navigateur.
- Remplis le formulaire et clique "Envoyer"
- Les commentaires apparaissent en dessous
- Ils persistent même après rechargement de la page (F5)
- Clique ✕ sur une carte pour supprimer un commentaire
- Clique "Effacer tous les commentaires" pour tout réinitialiser

## Outils recommandés

- **Éditeur** : Visual Studio Code + extension "Live Server"
- **Optimisation images** : squoosh.app
- **Icônes** : fonts.google.com/icons
